# EDC 隨身破窗尺
![image](https://hackmd.io/_uploads/Bkbcmg0F1g.png)


## 功能
- 能隨身上飛機的防身工具
- 有效測量長度10公分,使用雷雕不鏽鋼片,透過螺絲組合到本體
    - cm
    - inch
    - pcb footprint
- 本體全長12cm內
- 帶背夾
- 可纏傘繩
- 鎢鋼破窗頭(銷釘、頂針)
- 相容MOLLE
